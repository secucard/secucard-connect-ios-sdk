\subsection{Detailed Description}
The Rest\+Service\+Manager is a channel to connect to the secucard api in a standardized way as it conforms to the Service\+Manager\textquotesingle{}s protocol. The Rest\+Service\+Manager is also used to retrieve the auth token before being able to connect to stomp

Thr \hyperlink{interface_s_c_rest_service_manager}{S\+C\+Rest\+Service\+Manager} is responsible for any R\+E\+ST communication with the given server
