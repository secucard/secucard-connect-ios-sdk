Detailed Description

The RestServiceManager is a channel to connect to the secucard api in a standardized way as it conforms to the ServiceManager's protocol. The RestServiceManager is also used to retrieve the auth token before being able to connect to stomp

Thr SCRestServiceManager is responsible for any REST communication with the given server
